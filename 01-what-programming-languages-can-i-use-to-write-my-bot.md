### What programming languages can I use to write my bot?

Riddles.io currently supports the following programming languages:

- C
- C#
- C++
- D
- Go
- Haskell
- Java
- JavaScript
- Kotlin
- Lua
- OCaml
- Pascal
- Python 3
- Perl
- PHP
- Ruby
- Scala

However, note that not all languages might be available for any given
competition.
